# 🧠 MindScope: Mental Health & Lifestyle Predictor

## 📌 Overview

MindScope is an end-to-end Machine Learning project that analyzes lifestyle and behavioral patterns to predict mental health risk levels.

The system is built using real-world survey data and provides predictions through an interactive graphical user interface (GUI).

---

## 📊 Dataset

* Source: Global Mental Health & Lifestyle Survey
* Type: Tabular data (numerical + categorical)
* Includes:

  * Screen time
  * Sleep habits
  * Physical activity
  * Stress levels
  * Demographics

---

## ⚙️ Features

* Data preprocessing (handling missing values, encoding, scaling)
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine Learning model training
* Prediction system
* Interactive GUI for user input

---

## 🧠 Model

* Algorithms used:

  * Logistic Regression
  * Random Forest
  * (Optional) Neural Network
* Evaluation metrics:

  * Accuracy
  * Precision / Recall
  * Confusion Matrix

---

## 🖥️ GUI

Users can input their lifestyle data (sleep, screen time, activity, etc.) and receive a prediction of their mental health risk level.
